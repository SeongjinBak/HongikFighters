Hongik Fighters
===============
**홍익 파이터즈** 는 **2019년 4월 1일** 부터 **2019년 5월 15일**까지, 홍익대학교 게임동아리 **ExP Make**팀에서, 
기획자 2, 아티스트 3, 프로그래머 2, 사운드 1명의 참여로 만들어졌습니다.
-------------------------------------------------------------------------------------------
0. 플레이 동영상 
	- https://www.youtube.com/watch?v=ny6uW_Tfllc
1. 게임 정보
	- 2019년 홍익대학교 대동제의 수익사업을 위해 만들었습니다.
	- 미식축구 동아리 "Cowboys", 태권도 동아리 "홍익태권", 클래식 기타 동아리 "Nefer", 게임동아리 "ExP". 이상 4개 동아리와 협업 하였습니다.
	- 친구와 각각 동아리를 선택해서 격투 실력을 겨루어 보세요. 각 동아리의 특성을 엿볼수 있습니다. 
2. 개발 버전
	- Unity 2017.4.2f2 Personal (64bit)
	- Visual Studio 2017
3. 라이센스
	- 모든 권한은 **ExP Make**에서 가집니다. 
	- 모든 문의는 https://www.facebook.com/HongikGameClubExP/?ref=br_rs 에 해주세요.
	- ExP Make의 동의가 없는 Hongik Fighters의 이미지, 코드, 사운드등의 프로그램의 복제 및 사용은 **불허** 합니다.
4. 게임의 다운로드 및 실행
	- 다운로드는 https://github.com/SeongjinBak/HongikFighters/blob/master/HongikFighters.zip 에서 가능합니다.
	- 해상도는 1366 x 768 을 권장합니다. .exe파일 실행 후 Screen에서 설정할 수 있습니다.
